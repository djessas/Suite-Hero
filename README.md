# Heroes Project

Mini-projet JavaScript : gestion de super-héros.

## Fonctionnalités

- Chargement des héros depuis un fichier JSON
- Sauvegarde des héros dans le localStorage
- Affichage dynamique
- Ajout d’un nouveau héros via formulaire
- Suppression d’un héros
- Barre de chargement animée
- Recherche en temps réel
- Déploiement sur GitHub Pages
- Gestion du projet avec Git (branches par fonctionnalité)
